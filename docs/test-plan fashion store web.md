## 1. Introduction
This test plan defines the testing strategy, objectives and scope for the Fashionista online fashion store web application.

## 2. Objectives
- Validate core user flows (browse, add to cart, checkout).
- Confirm system stability and usability.
- Identify bugs and usability issues.
- Provide confidence in release readiness.

## 3. Scope
### In Scope
- Login & Registration
- Product browsing & filters
- Cart & Checkout
- UI & Responsiveness
- Basic validation & error handling

### Out of Scope
- Performance testing
- Load/Stress testing
- Deep security testing
- Mobile native app

## 4. Test Approach
Manual testing using:
- Scenario based testing
- Boundary value analysis
- Negative testing

## 5. Entry Criteria
- Build deployed
- Test environment stable
- Test data available

## 6. Exit Criteria
- All planned test cases executed
- No critical bugs open
- Test summary completed

## 7. Risks
- Environment instability
- Incomplete requirements

## 8. Deliverables
- Test Plan
- Test Cases
- Bug Reports
- Regression Checklist
- Test Summary
